Heroic DEB repository how-to

Run the following commands as root user:

- add repository key to Trusted software providers list

curl -s --compressed "https://deb.arrow-systems.de/heroicapt.gpg" | gpg --dearmor | tee /etc/apt/trusted.gpg.d/heroicapt.gpg > /dev/null

- add the repository:

curl -s --compressed -o /etc/apt/sources.list.d/heroicapt.list "https://deb.arrow-systems.de/heroicapt.list"

- update apt cache:

apt update

- install anydesk:

apt install heroic
